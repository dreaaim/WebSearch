# 可信联网搜索系统架构设计 v2

## 1. v1 问题回顾与 v2 改进概述

### 1.1 v1 架构局限性

v1 架构基于规则和关键词匹配进行信源分类与碰撞解决，存在以下核心局限性：

| 局限性 | 问题描述 | 影响 |
|--------|----------|------|
| **查询能力有限** | 依赖用户原始查询，无法理解深层意图，无法进行查询扩展 | 搜索结果可能遗漏重要信息源 |
| **结果筛选粗糙** | 基于 URL 规则匹配，无法处理无域名的内容源（如微信公众号、知乎专栏） | 大量优质内容被忽略 |
| **碰撞解决简单** | 使用 Jaccard 文本相似度，仅考虑词汇重叠，不理解语义 | 语义相似但措辞不同的 claim 可能被误判为冲突 |
| **缺乏重排序** | 原始搜索结果直接返回，未结合可信度权重 | 高可信信源可能排在低可信信源之后 |
| **黑白名单静态** | 规则需手动配置，无法运行时动态查询 | 无法应对新出现的信源 |

### 1.2 v2 核心改进

v2 版本通过引入 LLM 和 Embedding 模型，增强系统的语义理解能力：

```
v1 流程:                              v2 流程:
用户查询 ──→ 搜索 ──→ 规则筛选 ──→ Jaccard碰撞 ──→ 返回      │
     │              │              │              │         │
     │              │              │              │         │
     ▼              ▼              ▼              ▼         ▼
   [简单]        [搜索引擎]    [URL规则]     [词汇相似]   [无重排]

用户查询 ──→ 改写 ──→ 搜索 ──→ LLM筛选 ──→ Embedding碰撞 ──→ LLM裁决 ──→ 重排 ──→ 返回
     │         │         │         │              │            │          │
     ▼         ▼         ▼         ▼              ▼            ▼          ▼
   [意图理解] [多路发散] [增强语法] [语义分类]   [语义相似]    [智能判断]  [综合排序]
```

### 1.3 v2 架构总览

v2 架构在搜索引擎两侧分别增强：

- **左侧增强（查询侧）**: QueryRewriter - 意图理解、查询发散、查询增强
- **右侧增强（结果侧）**: LLMSourceClassifier - 信源提取、相关性判断、黑白名单工具调用
- **核心碰撞升级**: Embedding 语义相似度 + LLM Judge 智能裁决
- **结果优化**: Reranker 综合权重重排序

### 2.5 QueryRewriter 接口定义

```python
from typing import List, Optional
from dataclasses import dataclass
from enum import Enum

@dataclass
class QueryRewriteResult:
    original_query: str
    rewritten_queries: List[str]
    intent: str
    entities: List[str]
    time_range: Optional[str] = None

class QueryRewriter:
    def __init__(
        self,
        llm_client,
        intent_analyzer: IntentAnalyzer,
        query_expander: QueryExpander,
        query_enhancer: QueryEnhancer
    ):
        self.llm_client = llm_client
        self.intent_analyzer = intent_analyzer
        self.query_expander = query_expander
        self.query_enhancer = query_enhancer

    async def rewrite(self, query: str) -> QueryRewriteResult:
        """
        执行完整的查询改写流程

        Returns:
            QueryRewriteResult: 包含原始查询、改写后查询列表、意图分析结果
        """
        intent_result = await self.intent_analyzer.analyze(query)

        expanded_queries = await self.query_expander.expand(
            query, intent_result
        )

        enhanced_queries = await self.query_enhancer.enhance(
            expanded_queries, intent_result
        )

        return QueryRewriteResult(
            original_query=query,
            rewritten_queries=enhanced_queries,
            intent=intent_result.intent_type,
            entities=intent_result.key_entities,
            time_range=intent_result.time_range
        )
```

#### 3.4.2 Tool Call 使用流程

```python
class LLMSourceClassifier:
    def __init__(
        self,
        llm_client,
        source_extractor: LLMSourceExtractor,
        relevance_scorer: RelevanceScorer
    ):
        self.llm_client = llm_client
        self.source_extractor = source_extractor
        self.relevance_scorer = relevance_scorer

        # 注册 Tool Calls
        self.tools = [
            BlacklistChecker(),
            WhitelistChecker()
        ]

    async def classify(self, result: SearchResult, query: str) -> ClassifiedResult:
        # 步骤1: 提取信源信息
        source_info = await self.source_extractor.extract(result)

        # 步骤2: 判断相关性 (使用 Tool Call 查询黑白名单)
        classification = await self.llm_client.complete_with_tools(
            prompt=self._build_classification_prompt(result, query, source_info),
            tools=self.tools
        )

        # 步骤3: 综合分类结果
        return ClassifiedResult(
            result=result,
            source_info=source_info,
            classification=classification,
            relevance_score=classification.get("relevance_score", 5)
        )
```

```python
from dataclasses import dataclass
from typing import Optional, List

@dataclass
class SourceInfo:
    source_name: str
    source_type: SourceType
    source_domain: str
    author: Optional[str] = None
    is_verified: bool = False

@dataclass
class ClassifiedResult:
    result: SearchResult
    source_info: SourceInfo
    classification: Classification
    relevance_score: float
    whitelist_level: Optional[str] = None
    is_blacklisted: bool = False

class LLMSourceClassifier:
    def __init__(
        self,
        llm_client,
        source_extractor: LLMSourceExtractor,
        relevance_scorer: RelevanceScorer,
        blacklist_checker: BlacklistChecker,
        whitelist_checker: WhitelistChecker

        self.llm_client = llm_client
        self.source_extractor = source_extractor
        self.relevance_scorer = relevance_scorer
        self.blacklist_checker = blacklist_checker
        self.whitelist_checker = whitelist_checker

    async def filter_and_classify(
        self,
        results: List[SearchResult],
        query: str
    ) -> List[ClassifiedResult]:

        对搜索结果进行筛选和分类

        Returns:

        classified_results = []

            key=lambda x: x.relevance_score,
```

### 4.2 语义相似度计算

#### 4.2.1 余弦相似度公式

```
similarity(A, B) = cos(θ) = (A · B) / (||A|| × ||B||)
                 = Σ(a_i × b_i) / (√Σ(a_i²) × √Σ(b_i²))
```

#### 4.2.2 Embedding 相似度引擎实现

```python
import numpy as np
from typing import List, Tuple

class EmbeddingSimilarityEngine:
    def __init__(self, embedding_client):
        self.embedding_client = embedding_client

    async def compute_similarity(
        self,
        text1: str,
        text2: str
    ) -> float:
        """
        计算两个文本的语义相似度

        Returns:
            float: 相似度得分 (0-1)
        """
        embeddings = await self.embedding_client.encode([text1, text2])
        vec1, vec2 = embeddings[0], embeddings[1]

        return self._cosine_similarity(vec1, vec2)

    async def compute_batch_similarities(
        self,
        claims: List[Claim]
    ) -> List[List[float]]:
        """
        计算所有 claim 对之间的相似度矩阵

        Returns:
            n x n 相似度矩阵
        """
        texts = [claim.statement for claim in claims]
        embeddings = await self.embedding_client.encode(texts)

        n = len(claims)
        similarity_matrix = np.zeros((n, n))

        for i in range(n):
            for j in range(n):
                similarity_matrix[i][j] = self._cosine_similarity(
                    embeddings[i], embeddings[j]
                )

        return similarity_matrix.tolist()

    def _cosine_similarity(self, vec1: np.ndarray, vec2: np.ndarray) -> float:
        dot_product = np.dot(vec1, vec2)
        norm1 = np.linalg.norm(vec1)
        norm2 = np.linalg.norm(vec2)
        return dot_product / (norm1 * norm2)
```

### 4.3 混合相似度计算 (v2 增强)

v2 版本将 Jaccard 相似度与 Embedding 相似度结合：

```python
class HybridSimilarityEngine:
    def __init__(
        self,
        embedding_engine: EmbeddingSimilarityEngine,
        jaccard_weight: float = 0.3,
        embedding_weight: float = 0.7
    ):
        self.embedding_engine = embedding_engine
        self.jaccard_weight = jaccard_weight
        self.embedding_weight = embedding_weight

    async def compute_hybrid_similarity(
        self,
        claim1: Claim,
        claim2: Claim
    ) -> float:
        """
        计算混合相似度

        hybrid = α × jaccard + β × embedding
        """
        jaccard_sim = self._jaccard_similarity(
            claim1.statement,
            claim2.statement
        )

        embedding_sim = await self.embedding_engine.compute_similarity(
            claim1.statement,
            claim2.statement
        )

        return (
            self.jaccard_weight * jaccard_sim +
            self.embedding_weight * embedding_sim
        )

    def _jaccard_similarity(self, text1: str, text2: str) -> float:
        words1 = set(text1.lower().split())
        words2 = set(text2.lower().split())
        return len(words1 & words2) / len(words1 | words2)
```

```
- 类型: {source_type_b}
- 内容: {statement_b}
- 时间: {timestamp_b}

## 裁决标准 (按优先级排序)

1. **权威性**: 官方机构 > 知名媒体 > KOL > 个人
2. **时效性**: 最新的信息优先，但也要考虑信息的历史稳定性
3. **一致性**: 与其他可靠信源一致的结论更可信
4. **具体性**: 包含具体数据、来源的结论更可信
5. **安全性**: 涉及潜在风险的结论需要更高标准验证

    "warnings": ["警告1", "警告2"]
}

注意:
- 如果双方都有合理依据，选择更安全、更保守的结论
```

```python
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class CollisionJudgment:
    collision_id: str
    winner: str
    confidence: float
    reason: str
    safety_score: float
    ranking: List[str]
    consensus_level: str
    warnings: List[str]

class LLMCollisionJudge:
    def __init__(self, llm_client):
        self.llm_client = llm_client

    async def judge(
        self,
        collision: FactCollision,
        original_query: str
    ) -> CollisionJudgment:
        """
        对碰撞进行裁决

        Returns:
            CollisionJudgment: 包含裁决结果的详细输出
        """
        prompt = self._build_judgment_prompt(collision, original_query)
        response = await self.llm_client.complete(prompt)
        return self._parse_judgment(collision.collision_id, response)

    async def judge_batch(
        self,
        collisions: List[FactCollision],
        original_query: str
    ) -> List[CollisionJudgment]:
        """批量裁决"""
        judgments = []
        for collision in collisions:
            judgment = await self.judge(collision, original_query)
            judgments.append(judgment)
        return judgments
```

## 6. Reranker 重排序

### 6.1 综合权重计算

#### 6.1.1 权重因子定义

```python
class RerankConfig:
    DEFAULT_WEIGHTS = {
        "relevance": 0.4,       # 内容相关性权重
        "trustworthiness": 0.3, # 信源可信度权重
        "freshness": 0.15,      # 时效性权重
        "authority": 0.15       # 权威性权重
    }

class AuthorityLevel:
    NATIONAL_OFFICIAL = 1.0
    PROVINCIAL_OFFICIAL = 0.85
    MUNICIPAL_OFFICIAL = 0.7
    NATIONAL_MEDIA = 0.8
    PROVINCIAL_MEDIA = 0.65
    KOL_BIG = 0.6
    KOL_MEDIUM = 0.5
    KOL_SMALL = 0.4
    MEDIA = 0.45
    INDIVIDUAL = 0.3
```

#### 6.1.2 综合得分公式

```
FinalScore = w_relevance × RelevanceScore
           + w_trust × TrustworthinessScore
           + w_fresh × FreshnessScore
           + w_auth × AuthorityScore
```

其中：

- **RelevanceScore**: 来自 LLMSourceClassifier 的相关性评分 (0-10)
- **TrustworthinessScore**: 基于 Classification 的可信度 (WHITE=1.0, GRAY=0.5, BLACK=0.0)
- **FreshnessScore**: 基于发布时间衰减 (最近7天=1.0, 逐渐衰减)
- **AuthorityScore**: 基于 SourceType 和 SourceLevel 的权威性评分

### 6.2 重排序算法

#### 6.2.1 Reranker 实现

```python
class Reranker:
    def __init__(self, config: RerankConfig = None):
        self.config = config or RerankConfig()
        self.freshness_calculator = FreshnessCalculator()

    def rerank(
        self,
        results: List[ClassifiedResult],
        judgments: List[CollisionJudgment] = None
    ) -> List[ClassifiedResult]:
        """
        对结果进行重排序

        Args:
            results: 已分类的搜索结果
            judgments: 碰撞裁决结果 (可选)

        Returns:
            重排序后的结果列表
        """
        scored_results = []

        for result in results:
            score = self._calculate_composite_score(result, judgments)
            scored_results.append((result, score))

        # 按综合得分降序排序
        scored_results.sort(key=lambda x: x[1], reverse=True)

        return [result for result, _ in scored_results]

    def _calculate_composite_score(
        self,
        result: ClassifiedResult,
        judgments: List[CollisionJudgment]
    ) -> float:
        """计算综合得分"""

        # 1. 相关性得分 (0-1)
        relevance = result.relevance_score / 10.0

        # 2. 可信度得分 (0-1)
        trustworthiness = self._get_trustworthiness(result)

        # 3. 时效性得分 (0-1)
        freshness = self.freshness_calculator.calculate(
            result.result.published_date
        )

        # 4. 权威性得分 (0-1)
        authority = self._get_authority_score(result)

        # 5. 裁决调整 (如果该结果在碰撞中被裁决)
        if judgments:
            judgment_bonus = self._get_judgment_bonus(result, judgments)
        else:
            judgment_bonus = 0.0

        # 综合得分
        final_score = (
            self.config.weights["relevance"] * relevance +
            self.config.weights["trustworthiness"] * trustworthiness +
            self.config.weights["freshness"] * freshness +
            self.config.weights["authority"] * authority +
            judgment_bonus
        )

        return final_score

    def _get_trustworthiness(self, result: ClassifiedResult) -> float:
        if result.is_blacklisted:
            return 0.0
        if result.classification == Classification.WHITE:
            return 1.0
        if result.classification == Classification.GRAY:
            return 0.5
        return 0.0

    def _get_authority_score(self, result: ClassifiedResult) -> float:
        source_type = result.source_info.source_type
        source_level = result.source_info.source_level

        # 根据 source_type 和 source_level 计算权威性
        if source_type == SourceType.OFFICIAL:
            level_scores = {
                SourceLevel.NATIONAL: AuthorityLevel.NATIONAL_OFFICIAL,
                SourceLevel.PROVINCIAL: AuthorityLevel.PROVINCIAL_OFFICIAL,
                SourceLevel.MUNICIPAL: AuthorityLevel.MUNICIPAL_OFFICIAL,
                SourceLevel.LOCAL: 0.5
            }
            return level_scores.get(source_level, 0.5)

        if source_type == SourceType.MEDIA:
            level_scores = {
                SourceLevel.NATIONAL: AuthorityLevel.NATIONAL_MEDIA,
                SourceLevel.PROVINCIAL: AuthorityLevel.PROVINCIAL_MEDIA,
                SourceLevel.MUNICIPAL: 0.5,
                SourceLevel.LOCAL: 0.35
            }
            return level_scores.get(source_level, 0.35)

        if source_type == SourceType.KOL:
            # 简化处理，实际应根据粉丝数细分
            return AuthorityLevel.KOL_MEDIUM

        return AuthorityLevel.INDIVIDUAL


class FreshnessCalculator:
    def calculate(self, published_date: str) -> float:
        """计算时效性得分，0-1 之间"""
        if not published_date:
            return 0.5  # 无日期信息给予中间值

        try:
            pub_date = datetime.fromisoformat(published_date)
            days_old = (datetime.now() - pub_date).days

            if days_old <= 7:
                return 1.0
            elif days_old <= 30:
                return 0.8
            elif days_old <= 90:
                return 0.6
            elif days_old <= 365:
                return 0.4
            else:
                return 0.2
        except:
            return 0.5
```

## 7. SearchOrchestrator v2 流程

### 7.1 完整流程

```python
class SearchOrchestratorV2:
    def __init__(
        self,
        provider: SearchProvider,
        query_rewriter: QueryRewriter,
        source_classifier: LLMSourceClassifier,
        embedding_engine: EmbeddingSimilarityEngine,
        collision_judge: LLMCollisionJudge,
        reranker: Reranker,
        summary_generator: SummaryGenerator
    ):
        self.provider = provider
        self.query_rewriter = query_rewriter
        self.source_classifier = source_classifier
        self.embedding_engine = embedding_engine
        self.collision_judge = collision_judge
        self.reranker = reranker
        self.summary_generator = summary_generator

    async def search_with_trust_v2(
        self,
        query: str,
        options: SearchOptions = None
    ) -> TrustedSearchResult:
        """
        v2 可信搜索完整流程
        """

        # === Step 1: 查询改写 ===
        rewrite_result = await self.query_rewriter.rewrite(query)
        rewritten_queries = rewrite_result.rewritten_queries

        # === Step 2: 多路搜索 ===
        all_results = []
        for rewritten_query in rewritten_queries:
            response = await self.provider.search(rewritten_query, options)
            all_results.extend(response.results)

        # === Step 3: 结果筛选与分类 ===
        classified_results = await self.source_classifier.filter_and_classify(
            all_results, query
        )

        # === Step 4: Embedding 相似度计算 ===
        claims = [self.claim_extractor.extract(r.result) for r in classified_results]
        similarity_matrix = await self.embedding_engine.compute_batch_similarities(claims)

        # === Step 5: 碰撞检测与分组 ===
        collisions = self.collision_detector.detect_grouped(
            claims, similarity_matrix
        )

        # === Step 6: LLM 碰撞裁决 ===
        judgments = await self.collision_judge.judge_batch(
            collisions, query
        )

        # === Step 7: Reranker 重排序 ===
        reranked_results = self.reranker.rerank(
            classified_results, judgments
        )

        # === Step 8: 摘要生成 ===
        summary, consensus_facts, disputed_facts = await self.summary_generator.generate(
            reranked_results, collisions
        )

        # === Step 9: 组装结果 ===
        return TrustedSearchResult(
            query=query,
            response=SearchResponse(
                query=query,
                results=[r.result for r in reranked_results],
                total_count=len(reranked_results),
                search_time=0
            ),
            classified_results=self._group_by_classification(reranked_results),
            collisions=collisions,
            judgments=judgments,
            summary=summary,
            consensus_facts=consensus_facts,
            disputed_facts=disputed_facts,
            metadata={
                "version": "v2",
                "rewrite_result": rewrite_result,
                "rerank_weights": self.reranker.config.weights
            }
        )
```
