# v2 版本使用指南

## 概述

v2 版本在 v1 基础上引入了 LLM 和 Embedding 模型，增强了查询改写、信源分类、信息碰撞和结果排序能力。

## 核心特性

### 1. 查询改写 (QueryRewriter)
- 意图理解：自动分析用户查询的语义和目的
- 查询发散：生成多个不同角度的查询方向
- 查询增强：补充 site:、时间范围等搜索引擎高级语法

### 2. LLM 信源分类 (LLMSourceClassifier)
- 从 title/content 提取信息源（非依赖 URL）
- 判断结果与查询意图的相关性
- 通过 Tool Call 动态查询黑白名单

### 3. Embedding 语义相似度
- 引入 Embedding 模型计算 claim 之间的语义相似度
- 支持 Jaccard + Embedding 混合相似度计算

### 4. LLM 碰撞裁决
- 对碰撞结果进行智能判断
- 选择相对安全的结果
- 提供裁决理由和置信度

### 5. Reranker 重排序
- 综合考虑相关性、可信度、时效性、权威性
- 对搜索结果进行综合重排序

## 快速开始

### 安装依赖

```bash
pip install openai>=1.0.0
pip install numpy>=1.24.0
```

### 基本用法

```python
from web_search.core.orchestrator_v2 import SearchOrchestratorV2
from web_search.providers.searxng import SearXNGProvider
from web_search.classifier.source_classifier import SourceClassifier
from web_search.rewriter import QueryRewriter
from web_search.reranker import Reranker

# 创建 v2 编排器
orchestrator = SearchOrchestratorV2(
    provider=SearXNGProvider(base_url="http://localhost:8080"),
    source_classifier=SourceClassifier(),
    query_rewriter=QueryRewriter(),  # v2 查询改写
    reranker=Reranker(),              # v2 重排序
    use_v2_features=True
)

# 执行搜索
result = orchestrator.search_with_trust("最近 AI 领域有什么最新突破？")

print(f"查询: {result.query}")
print(f"结果数: {len(result.response.results)}")
print(f"白名单结果: {len(result.classified_results['white'])}")
print(f"灰名单结果: {len(result.classified_results['gray'])}")
print(f"黑名单结果: {len(result.classified_results['black'])}")
print(f"碰撞数: {len(result.collisions)}")
```

### 使用 LLM 增强功能

```python
from openai import AsyncOpenAI

# 创建 LLM 客户端
llm_client = AsyncOpenAI(api_key="your-api-key")

# 创建带 LLM 增强的组件
from web_search.rewriter import QueryRewriter
from web_search.classifier import LLMSourceClassifier, RelevanceScorer
from web_search.resolver import LLMCollisionJudge

query_rewriter = QueryRewriter(llm_client=llm_client)
llm_classifier = LLMSourceClassifier(
    llm_client=llm_client,
    relevance_scorer=RelevanceScorer(llm_client=llm_client)
)
collision_judge = LLMCollisionJudge(llm_client=llm_client)

# 创建 v2 编排器
orchestrator = SearchOrchestratorV2(
    provider=SearXNGProvider(base_url="http://localhost:8080"),
    source_classifier=SourceClassifier(),
    query_rewriter=query_rewriter,
    llm_classifier=llm_classifier,
    collision_judge=collision_judge,
    use_v2_features=True
)

# 执行异步搜索
result = await orchestrator.search_with_trust_async(
    "特斯拉最新股价是多少？"
)
```

### 使用 Embedding 相似度

```python
from web_search.resolver import EmbeddingSimilarityEngine, HybridSimilarityEngine
from openai import AsyncOpenAI

# 创建 Embedding 客户端（使用 OpenAI）
embedding_client = AsyncOpenAI(api_key="your-api-key")

# 或者使用本地 Embedding 模型
# from your_embedding_module import LocalEmbeddingClient
# embedding_client = LocalEmbeddingClient(model_path="/models/bge-large-zh")

# 创建 Embedding 相似度引擎
embedding_engine = EmbeddingSimilarityEngine(embedding_client=embedding_client)

# 创建混合相似度引擎（Jaccard + Embedding）
hybrid_engine = HybridSimilarityEngine(
    embedding_engine=embedding_engine,
    jaccard_weight=0.3,
    embedding_weight=0.7
)

# 在编排器中使用
orchestrator = SearchOrchestratorV2(
    provider=SearXNGProvider(base_url="http://localhost:8080"),
    source_classifier=SourceClassifier(),
    embedding_engine=hybrid_engine,  # 注入混合相似度引擎
    use_v2_features=True
)
```

### 配置权重

```python
from web_search.reranker import Reranker, RerankConfig

# 自定义权重配置
config = RerankConfig(weights={
    "relevance": 0.5,        # 提高相关性权重
    "trustworthiness": 0.3,   # 保持可信度权重
    "freshness": 0.1,       # 降低时效性权重
    "authority": 0.1         # 降低权威性权重
})

reranker = Reranker(config=config)

orchestrator = SearchOrchestratorV2(
    provider=SearXNGProvider(base_url="http://localhost:8080"),
    source_classifier=SourceClassifier(),
    reranker=reranker,
    use_v2_features=True
)
```

## 模块说明

### rewriter - 查询改写模块

| 类 | 说明 |
|---|---|
| `QueryRewriter` | 查询改写器主类 |
| `IntentAnalyzer` | 意图分析器 |
| `QueryExpander` | 查询发散器 |
| `QueryEnhancer` | 查询增强器 |

### classifier - 分类器模块

| 类 | 说明 |
|---|---|
| `SourceClassifier` | v1 规则分类器 |
| `LLMSourceClassifier` | v2 LLM 分类器 |
| `RelevanceScorer` | 相关性评分器 |

### resolver - 碰撞解决模块

| 类 | 说明 |
|---|---|
| `FactResolver` | v1 碰撞解决器 |
| `EmbeddingSimilarityEngine` | Embedding 相似度引擎 |
| `HybridSimilarityEngine` | 混合相似度引擎 |
| `LLMCollisionJudge` | LLM 碰撞裁决器 |

### reranker - 重排序模块

| 类 | 说明 |
|---|---|
| `Reranker` | 结果重排序器 |
| `FreshnessCalculator` | 时效性计算器 |

## 迁移指南

### 从 v1 迁移到 v2

v1 和 v2 可以共存，无需破坏性修改：

```python
# v1 用法（保持不变）
from web_search.core.orchestrator import SearchOrchestrator

orchestrator_v1 = SearchOrchestrator(
    provider=provider,
    source_classifier=classifier
)
result = orchestrator_v1.search_with_trust(query)

# v2 用法（新增功能）
from web_search.core.orchestrator_v2 import SearchOrchestratorV2

orchestrator_v2 = SearchOrchestratorV2(
    provider=provider,
    source_classifier=classifier,
    query_rewriter=QueryRewriter(),
    reranker=Reranker(),
    use_v2_features=True
)
result = orchestrator_v2.search_with_trust(query)
```
