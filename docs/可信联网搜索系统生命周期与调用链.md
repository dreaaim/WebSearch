## 一、系统架构演进说明

### 1.1 v1 / v2 / v3 核心差异对比

| 维度 | v1 架构 | v2 架构 | v3 架构 |
|------|---------|---------|---------|
| **查询处理** | 用户原始查询直接搜索 | QueryRewriter 意图理解 + 查询发散 | QueryRewriter 保持 + 增强查询语法 |
| **结果初筛** | URL 规则匹配 + 黑白名单 | LLM SourceClassifier 语义分类 | HybridFilterEngine (Embedding + BM25) |
| **信源可信度** | 静态黑白名单 | 静态黑白名单 + Tool Call 查询 | TrustRankLadder ELO 动态评分 |
| **碰撞检测** | Jaccard 文本相似度 | Embedding 语义相似度 + LLM Judge | OrthogonalCollisionDetector (NLI/SPO/数值三维度) |
| **结果排序** | 原始搜索结果直接返回 | Reranker 综合权重重排 | MultiFactorReranker (时效性 + 相关性 + 可信度) |
| **事实理解** | 无 | Claim 提取 | FactExtractor (SPO/NLI/数值/时间多维提取) |
| **摘要生成** | SummaryGenerator 聚类摘要 | SummaryGenerator 保持 | 移至最终可信事实输出 |
| **核心问题** | LLM 成本高、静态名单无法适应新信源 | 缺乏结构化事实理解 | 多维度碰撞检测 + LLM 复核机制 |

## 五、数据模型流转说明

### 5.1 核心数据模型定义

#### 5.1.1 SearchResult (搜索结果)

```python
@dataclass
class SearchResult:
    title: str                      # 标题
    url: str                        # 链接
    snippet: str                    # 摘要/片段
    source_name: str                # 信源名称
    source_domain: str               # 信源域名
    source_type: SourceType          # 信源类型
    source_level: SourceLevel        # 信源层级
    published_date: Optional[str]    # 发布日期
    relevance_score: float = 0.0     # 相关性得分
    classification: Classification   # 分层结果
```

#### 5.1.2 HybridFilterResult (混合过滤结果)

```python
@dataclass
class HybridFilterResult:
    result: SearchResult              # 原始搜索结果
    embedding_score: float           # Embedding 相似度得分
    bm25_score: float                # BM25 词频得分
    hybrid_score: float               # 混合综合得分
```

#### 5.1.3 ContentFetchResult (内容获取结果)

```python
@dataclass
class ContentFetchResult:
    result: SearchResult              # 原始搜索结果
    content: str                      # 提取的正文内容
    fetch_success: bool               # 获取是否成功
    error_reason: Optional[str]       # 失败原因
```

#### 5.1.4 ExtractedFact (提取的事实)

```python
@dataclass
class ExtractedFact:
    fact_id: str                       # 事实ID
    statement: str                     # 事实陈述
    spo_triple: Optional[SPOTriple]      # SPO 三元组
    nli_label: Optional[str]            # NLI 标签
    numeric_values: List[NumericValue]  # 数值信息
    datetime_values: List[DatetimeValue]# 时间信息
    confidence_score: float             # 置信度得分
    confidence_reason: str              # 置信度理由
    source_domain: Optional[str]        # 来源域名
    trust_score: float = 0.0           # 可信度得分
```

#### 5.1.5 FactBucket (事实桶)

```python
@dataclass
class FactBucket:
    bucket_id: str                      # 桶ID
    facts: List[ExtractedFact]         # 桶内事实列表
    cluster_embedding: List[float]      # 聚类中心向量
```

#### 5.1.6 CollisionResult (碰撞检测结果)

```python
@dataclass
class CollisionResult:
```
